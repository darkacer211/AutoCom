AutoComm – Email & WhatsApp Automation Bot

AutoComm is a Python-based automation system designed to send emails, automate WhatsApp messages, and perform rule-based auto-replies with scheduling support.

🚀 Planned Features

Email sending and inbox monitoring

Rule-based auto-reply system

WhatsApp Web automation

Scheduled messages and tasks

Contact and log management using a database

🛠 Tech Stack

Python

Selenium

SMTP / IMAP

SQLite (planned)

📌 Project Status

Currently in development. This project focuses on building a modular and extensible automation framework.
