# MScheduler

A comprehensive open-source scheduling and automation platform with Email, WhatsApp, and AI integration. Built with Flask backend and Vue.js frontend.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.8+-green)
![License](https://img.shields.io/badge/license-MIT-blue)

## ?? Features

- **Email Automation**: Send emails, monitor inbox, and automate replies
- **WhatsApp Integration**: Automate WhatsApp messages with browser automation
- **Smart Scheduling**: Schedule tasks and messages with APScheduler
- **AI-Powered Analysis**: Email analysis with Gemini AI engine
- **Database Management**: SQLAlchemy ORM with comprehensive models
- **REST API**: Modern Flask REST API with JWT authentication
- **Interactive Dashboard**: Vue.js dashboard with Tailwind CSS styling
- **Alert System**: Smart alert system for WhatsApp notifications
- **Email Bot**: Advanced email reader and auto-reply system

## ?? Quick Start

### Backend Setup
```bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
cd web && python app.py
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

**Access**: Frontend at http://localhost:5173, Backend at http://localhost:5000/api
